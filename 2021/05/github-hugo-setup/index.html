My notes - elaborate

install git powershell module - install-module git I actually use the Github desktop to keep my repos in sync. This will work on multiple systems. https://desktop.github.com/

install hugo - choco install hugo -confirm install hugo extended - choco install hugo-extended -confirm https://gohugo.io/getting-started/installing/ - downloaded windows zip file and extracted the files into a Hugo folder, I set a path pointing to Hugo/bin to run the hugo.exe

copy public file to published website repo copy build files to another repo to allow access from multiple systems. used mklink /J to have files stored on OneDrive. I did not test this wiht the github desktop repo. I simply copy files after I am done testing. mklink /J C:\LinkToFolder C:\Users\Name\OriginalFolder

Created a github site - https://dpsmith005.github.io/ https://github.com/dpsmith005/ https://github.com/dpsmith005/dpsmith005.github.io/ https://github.com/dpsmith005/SCOMAlertshttps://github.com/dpsmith005/SCOMAlerts

Hugo

Build web site hugo new site <site_name> hugo –config .\config.toml

Run Hugo server hugo server hugo server -D (includes draft pages) in a browser use localhost:1313

in site folder create new file. will be placed in content folder hugo new a.md hugo new dir1/b.md hugo new dir1/dir2/c.md
